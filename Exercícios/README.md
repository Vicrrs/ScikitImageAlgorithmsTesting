1. **Segmentação de Imagens com Watershed**:
   - Objetivo: Implementar a segmentação de uma imagem complexa usando a técnica de watershed.
   - Desafio: Aplicar pré-processamento adequado para destacar fronteiras entre objetos.

2. **Restauração de Imagem com Deconvolução de Wiener**:
   - Objetivo: Restaurar uma imagem borrada usando deconvolução de Wiener.
   - Desafio: Estimar o PSF (Point Spread Function) apropriado para a imagem borrada.

3. **Detecção de Cantos com o Detector de Harris**:
   - Objetivo: Identificar cantos em uma imagem urbana ou de arquitetura.
   - Desafio: Ajustar parâmetros para maximizar a precisão na detecção de cantos reais.

4. **Reconhecimento de Padrões com Transformada de Hough**:
   - Objetivo: Detectar formas geométricas (como círculos ou linhas) em imagens.
   - Desafio: Lidar com ruído e variações de forma nas imagens de entrada.

5. **Análise de Textura com Co-ocorrência de Matrizes de Matiz**:
   - Objetivo: Classificar imagens baseadas em suas texturas.
   - Desafio: Selecionar e ajustar parâmetros de matrizes de co-ocorrência para diferentes texturas.

6. **Extração de Características com SIFT (Scale-Invariant Feature Transform)**:
   - Objetivo: Extrair e comparar características SIFT de duas imagens semelhantes.
   - Desafio: Implementar correspondência de características robusta a mudanças de escala e rotação.

7. **Segmentação de Imagens com SLIC Superpixels**:
   - Objetivo: Implementar segmentação de imagem usando SLIC superpixels.
   - Desafio: Ajustar o número de superpixels e a compactação para obter segmentação eficaz.

8. **Morfologia Matemática para Análise de Estruturas Celulares**:
   - Objetivo: Utilizar operações morfológicas para analisar imagens de células.
   - Desafio: Escolher e aplicar operações morfológicas apropriadas para destacar características celulares.

9. **Detecção de Objetos com Filtragem Adaptativa**:
   - Objetivo: Implementar detecção de objetos específicos em imagens usando filtros adaptativos.
   - Desafio: Projetar um filtro que seja eficaz mesmo em condições variáveis de iluminação e fundo.

10. **Análise de Movimento com Fluxo Óptico**:
    - Objetivo: Estimar e visualizar o movimento em uma sequência de imagens.
    - Desafio: Implementar uma solução robusta para lidar com movimentos rápidos ou complexos.
